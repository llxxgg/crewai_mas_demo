# 项目需求 - Manager 演示输入

## 需求描述

XiaoPaw 需要新增「智能日程助手」功能，帮助用户通过自然语言管理日历事件。

具体功能包括：
1. 用户用中文说「明天下午3点开产品评审会」，系统自动解析时间和事件并创建日历条目
2. 用户询问「本周还有哪些会议」，系统列出当周所有日程
3. 用户说「取消周五的那个会」，系统找到对应事件并删除

## 约束条件

- 上线时间：2周内
- 技术栈：Python + 飞书日历 API
- 用户界面：飞书聊天窗口（已有集成）

## 验收标准

- 自然语言解析准确率 ≥ 90%（基于10个测试用例）
- 日历操作成功率 100%（创建/查询/删除）
- 响应时间 ≤ 3秒
