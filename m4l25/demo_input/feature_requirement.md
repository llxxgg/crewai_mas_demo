# 功能需求 - Dev 演示输入

## 任务信息

- **任务ID**：T-01
- **任务名称**：自然语言日程解析模块
- **分配角色**：Dev
- **来源**：Manager 任务拆解（project_requirement.md）

## 功能描述

实现一个 NLP 解析模块，将用户的中文自然语言输入转化为结构化的日历事件对象。

**输入示例**：
- "明天下午3点开产品评审会"
- "下周一上午10点和张总会面，地点在A栋302"
- "这个周五下午2点到4点代码评审"

**输出格式**：
```python
{
    "title": "产品评审会",
    "start_time": "2026-03-27T15:00:00+08:00",
    "end_time": "2026-03-27T16:00:00+08:00",  # 默认1小时
    "location": "",
    "attendees": []
}
```

## 验收标准

1. 解析准确率 ≥ 90%（在10个测试用例上验证）
2. 支持相对时间（明天、下周一、这个周五）
3. 支持时间范围（2点到4点）
4. 地点和参与人为可选字段，解析失败时返回空字符串/空列表，不报错
5. 单次解析耗时 ≤ 1秒（不含网络请求）
6. 提供完整单元测试，覆盖正常路径 + 3个边界用例
