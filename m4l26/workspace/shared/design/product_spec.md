# 产品规格文档 - XiaoPaw 宠物健康记录功能

## 1. 产品概述

为 XiaoPaw 宠物管理应用新增健康记录模块，支持宠物主人独立、安全、持久地管理每只宠物的疫苗接种与就医历史，通过智能提醒提升宠物健康管理主动性与依从性。

## 2. 用户故事

- 作为宠物主人，我希望为爱宠添加疫苗接种记录（含疫苗名、接种日、下次到期日），以便建立完整的免疫档案；
- 作为宠物主人，我希望在宠物主页看到临近到期（30天内）的疫苗提醒标识，以便及时安排补种；
- 作为宠物主人，我希望添加就医记录（含日期、医院、原因、诊断、费用及备注），以便追溯诊疗历程；
- 作为宠物主人，我希望按日期倒序查看所有就医记录，以便快速定位最新就诊情况；
- 作为宠物主人，我希望每只宠物的健康数据相互隔离，避免信息混淆。

## 3. 功能规格

| 编号 | 模块 | 描述 | 优先级 |
|------|------|------|--------|
| F-01 | 疫苗记录 | 支持添加、查看疫苗记录；到期前30天在宠物主页显示醒目提醒标识 | P0 |
| F-02 | 就医记录 | 支持添加（含500字备注）、按日期倒序查看就医记录列表 | P0 |

## 4. 验收标准

1. 每只宠物拥有独立健康数据空间，不同宠物记录完全隔离；
2. 所有疫苗与就医记录均持久化存储，App 重启后数据完整保留；
3. 疫苗到期提醒逻辑准确：仅当「下次到期日」距当前日期 ≤30 天时，在宠物主页展示提醒标识；
4. 就医记录列表默认按「就诊日期」降序排列，最新记录始终置顶；
5. 产品文档包含且仅包含：产品概述、用户故事、功能规格、验收标准四部分。
